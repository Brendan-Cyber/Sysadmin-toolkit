This project is a collection of small system administration tools written in Bash and Python for learning purposes.

It focuses on Linux fundamentals, automation, and practical sysadmin workflows such as backups, file handling, system inspection, and scripting automation.

The goal is not production software, but hands-on learning of Linux internals, shell scripting, and Python automation.

This project is for educational purposes only.

Not production-ready
Not guaranteed to be safe in all edge cases
Use at your own risk
Designed for experimentation and learning, not deployment in critical environments

#Tools include (for now)

1. Backup Tool (Bash)

A simple backup and restore utility using tar and sha256sum.

Features:

Create compressed backups of directories
Generate SHA256 checksums
Verify backup integrity before restore
Restore archived directories

Limitations:

Basic error handling
May include edge cases not covered (e.g., recursive backups, path handling quirks)
Intended for learning, not large-scale backup management
